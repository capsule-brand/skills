What’s a “skill”?

A skill is a small tool you add to the Claude app. Once it’s added, Claude knows how to do one job really well — the Capsule way — without you explaining it every time.

Think of it like teaching Claude a new trick once, then just asking for it whenever you need it.

  • Download a skill below — it’s one small file.
  • Add it to Claude one time.
  • Just ask in plain English — Claude takes it from there.

Adding a skill

Four steps, about two minutes. No setup — if you can download a file, you can do this.

  1. 1
    Download the skill Pick any skill below and click its Download button. You’ll get a small .skill file — that’s the whole skill in one file. Note where it saves (usually your Downloads folder).
  2. 2
    Open Claude → Settings → Cowork In the Claude app, open Settings, then choose in the sidebar. This is the page where your skills live.
  3. 3
    Upload the file Click Upload skill and choose the .skill file you just downloaded. Claude adds it to your list — that’s it, the skill is ready.
  4. 4
    Use it — just ask Start a normal chat and say what you want in plain words. Each skill below shows an example to try. Prefer a ? Type its /slash-command instead.
