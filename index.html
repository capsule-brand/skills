Capsule Skills

Internal Claude skills for the Capsule team. Download one, add it to the Claude app, then just talk to it.

 

How to install any skill

  1. Download the skill below (a .skill file).
  2. Open the Claude app and go to Settings → Claude Cowork → Upload skill.
  3. Pick the file you just downloaded. The skill now shows up in your skills list, and you use it by describing what you want or typing its slash command.
